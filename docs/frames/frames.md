# Frames 

In Frame Tab we can add, delete and save frames.

Frame types :
1. CAN Frame
2. LIN Sporadic Frames
3. LIN Unconditional Frames
4. LIN Event Triggered Frames
5. FlexRay Frames
6. Ethernet Frames
